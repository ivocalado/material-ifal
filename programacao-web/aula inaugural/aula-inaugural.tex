\section{Objetivos}
\begin{frame}{Objetivos}
\begin{block}{Objetivo Geral}
 Compreender a aplicação das técnicas de programação no ambiente web, bem como as principais linguagens em utilização.
\end{block}
Específicos:
\begin{itemize}
  \item Aprender os principais conceitos e desenvolver programas em linguagens comumentes usadas para aplicações web, como xml, javascript, jsp e servlets.
  \item Utilizar a arquitetura cliente-servidor em sistemas web;
  \item Aprender a projetar e gerenciar projetos de aplicações web;
\end{itemize}
\end{frame}
